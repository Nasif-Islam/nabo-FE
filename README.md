# Nabo - Frontend

A social news aggregation web application built with React. This project interacts with a RESTful API to allow users to browse, vote on, and comment on articles.

## Features

- [ ] View all articles
- [ ] Filter articles by topic
- [ ] Sort articles by date, comment count, or votes
- [ ] View individual articles and their comments
- [ ] Vote on articles

## Tech Stack

- **Language:** JavaScript (May refactor to TypeScript)
- **Framework:** React (using Vite)
- **Styling:** CSS (Material UI)
